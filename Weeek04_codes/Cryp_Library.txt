1. PyCryptodome (self-contained Python library)
- No external C library is required.

2. OpenSSL
external C library: OpenSSL(libssl + libcrypto)
Your Python code
     │
PyOpenSSL (Python wrapper)
     │
OpenSSL C library (libssl + libcrypto)


3. liboqs-python(Post-quantum Cryptography)
external C library: liboqs
Your Python code
   │
liboqs-python (Python wrapper)
   │
liboqs (C/C++ library)
